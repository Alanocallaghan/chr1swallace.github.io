Autoimmunity and statistical genetics

This is a set of pages to store the code I use and related information in my statistical genetic analyses. For the corporate webpage with publications, please go to http://www-gene.cimr.cam.ac.uk/staff/wallace.

Information that exists here falls under:

Code and data related to papers

Bowes et al (2015) "Dense genotyping of immune-related susceptibility loci reveals new insights into the genetics of psoriatic arthritis"

R script for fine mapping https://github.com/chr1swallace/finemap-psa RA p values required for input were downloaded from ImmunoBase.

T1D ImmunoChip paper (in press)

R code used to perform numerical calculations exploring the appropriate p value thresholds for calling genetic association in one disease given genomewide significant genetic association at the same SNP in a related disease. https://github.com/chr1swallace/condpp.

T1D ImmunoChip Data

Liley and Wallace (2015) "A Pleiotropy-Informed Bayesian False Discovery Rate Adapted to a Shared Control Design Finds New Disease Associations From GWAS Summary Statistics"

Data Availability: GWAS summary statistics (p values) were downloaded from the ImmunoBase website (http://www.immunobase.org). Also uses T1D ImmunoChip Data for simulations to examine properties of the method.

James' code is on https://github.com/jamesliley/cFDR-commo%E2%80%8Bn-controls.

Wallace et al "Dissection of a complex disease susceptibility region using a Bayesian stochastic search approach to fine mapping" bioRxiv preprint

Uses the R package GUESSFM. For detailed pipeline of steps to run GUESSFM, see the vignettes.

T1D ImmunoChip Data

MS data was accessed through application to the International Multiple Sclerosis Genetic Consortium (IMSGC).
